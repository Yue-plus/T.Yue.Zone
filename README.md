# T.Yue.Zone

悦域短链接服务。
正在开发……

## 计划

### 3、7、30、180 天有效

```text
URL https://T.Yue.Zone/123ab
                       -----
                       ↑随机生成
```

### 长期有效

```text
URL https://123ab.T.Yue.Zone
            -----
            ↑可自定义，使用 DNS 显性 URL 记录
```
